# 插圖風格指引

> 任何生圖 / 素材任務**必讀**本檔,防止風格漂移。本檔是風格單一事實來源;各題主體與色組語意在 `subdocs/題庫/<id>.md` 定稿表。
> 風格定案:2026-07-16(基於 demo 封面、latte / teapot、daily-life 貓 / 狗 五張已採用圖)。

## 1. 風格核心(不可變)

- **一句話**:可愛溫馨的遊戲插畫 —— kawaii 但乾淨、不幼稚,storybook game art。
- **造型**:圓潤形狀、厚而柔的描邊、扁平上色 + 淡淡柔陰影。
- **色彩**:pastel 柔和色調。「粉色系」= pastel,**不是整張粉紅**;每張圖指定暖或冷一組:
  - **暖色組**:cream, soft peach, milky beige
  - **冷色組**:soft mint green, sage, pale aqua
- **構圖(選項圖)**:單一主體大而置中、佔滿大部分畫面,縮小到卡片尺寸仍一眼可辨;背景極簡素面 + gentle vignette;道具最多一個小件。
- **禁止**:任何文字 / 字母 / 浮水印、寫實照片感、3D render、強烈陰影、霓虹飽和色、雜亂背景、人臉。

## 2. 選項圖 prompt 模板(4:3)

【】內為逐圖替換槽,其餘照抄:

```
Cute cozy game illustration, one single centered subject: 【主體描述,
含姿態/情緒/材質等 1-3 個細節】. The subject fills most of the frame,
large and instantly recognizable even at small size. Simple minimal
background in 【warm pastel tones (cream, soft peach, milky beige) /
cool pastel tones (soft mint green, sage, pale aqua)】, at most one
small prop such as 【一個小道具,可省略】, no clutter, soft plain
backdrop with gentle vignette. Rounded shapes, thick soft outlines,
flat illustration with subtle soft shading, kawaii style but clean and
not childish. Storybook game art style. If any animal appears it must
be a tiny background accessory, not prominent. No text, no letters,
no watermark. 4:3 aspect ratio.
```

**NEGATIVE(固定)**:

```
text, letters, typography, watermark, logo, realistic photo, 3d render,
harsh shadows, saturated neon colors, cluttered background, human faces,
large animals, split screen
```

### 動物為主體的題目(如貓派 / 狗派)

- 模板中「If any animal appears it must be a tiny background accessory, not prominent.」**整句移除**。
- NEGATIVE 中 `large animals` 改為 `multiple animals`。

## 3. 封面 prompt 模板(16:9)

```
Cute cozy game illustration, split-screen composition divided by a soft
diagonal line, clear color contrast between the two halves:
LEFT half in warm pastel tones (cream, soft peach, milky beige) — 【暖側
場景描述】.
RIGHT half in cool pastel tones (soft mint green, sage, pale aqua) —
【冷側場景描述】.
Soft pastel color palette overall, the two sides clearly different in
hue but harmonious together. Rounded shapes, thick soft outlines, flat
illustration with subtle soft shading, kawaii style but clean and not
childish. Warm comfortable atmosphere, storybook game art style.
No text, no letters, no watermark. 16:9 aspect ratio.
```

**NEGATIVE**:同選項圖固定 negative,去掉 `split screen`(封面本來就是分割構圖)、`large animals`(封面動物依下方配件規則自律)。

## 4. 成對與色組規則

- **色組跟語意走**:每題兩張必為一暖一冷,暖冷歸屬依題庫定稿表「色組(語意)」欄,**不死綁 A=暖**(例:睡到自然醒=冷的夜感、早睡早起=暖的晨光)。
- **同題兩張同一輪連續生成**,確保筆觸與質感成對;風格對不上就整題重生,不單換一張。
- 動物僅在其為題目主體時可佔大畫面,否則只能當小配件(畫面佔比必須小)。

## 5. 產出與落地流程

1. 原始輸出存 `assets/generated/<題庫id>/`,檔名 `option-<主體>-<yyyymmddHHMM>.jpeg` / `cover-<描述>-<yyyymmddHHMM>.jpeg`,時間戳後綴避免覆蓋。
2. 每張圖旁存同名 `.prompt.txt`:第一行寫用途(題庫、題目、A/B、色組、比例),接 `PROMPT:` 與 `NEGATIVE:` 全文;已知瑕疵與接受決定也記在檔尾。
3. 正式採用的圖轉 **WebP 30–50KB**(選項圖 4:3 或正方形,封面 16:9),放入前端 `public/img/<題庫id>/`,路徑與題庫 JSON 一致。
4. `assets/generated/` 內的原始檔與 prompt 檔保留不刪,作為重生 / 追溯依據。

## 6. 已知陷阱(歷史教訓)

- 茶壺等懸空未落桌:主體是放置物時,prompt 加 `resting on the table` 之類落地描述(teapot 首版接受此瑕疵,重生時修正)。
- 「粉色系」誤解為整張粉紅:一律以暖 / 冷 pastel 色組指定,不用 pink 字眼。
